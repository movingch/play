# 움공플레이

움직이는교회의 예배, 수련회, 소그룹, 공동체 모임을 위한 레크리에이션 웹앱입니다.

## 실행 방법

```bash
npm install
npm run dev
```

## 배포 전 빌드 확인

```bash
npm run build
```

## Vercel 설정

- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`
