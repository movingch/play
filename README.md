# 움공플레이 v1.2

원본 이음플레이 스타일을 기준으로 다시 정리한 교회 레크리에이션 웹앱입니다.

## 포함 게임
- 스톱워치
- 타이머
- 주사위 / 윷놀이
- 랜덤 뽑기
- 빙고
- 단어 릴레이
- 문장 매칭
- 초성 게임
- 사진 맞추기
- 성경 넌센스 배틀

## 실행
```bash
npm install
npm run dev
```

## 배포
Vercel 설정:
- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist
- Install Command: npm install
